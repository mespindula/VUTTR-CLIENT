# VUTTR-CLIENT
front-end para a aplicação VUTTR

# Tecnologias
    React 16.13.1
    react-router-dom 5.2.0
    axios 0.19.2
    bootstrap 4.5.0
    
# Setup React.js
    npm install bootstrap
    npm install react-router-dom
    npm install axios

# Docker
	https://hub.docker.com/r/mespindula/vuttr-cliente
    run -it -p 3001:3000 mespindula/vuttr-cliente
